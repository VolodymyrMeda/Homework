add(item)¶

clear()¶

peek()¶

Returns the item at the front of the queue. Precondition: the queue is not empty. Raises: KeyError if the stack is empty

pop()¶

Removes and returns the item at the front of the queue. Precondition: the queue is not empty. Raises: KeyError if the queue is empty. Postcondition: the front item is removed from the queue

remove(index)¶

Removes and returns the item at index, where index ranges from 0 (the front) to size - 1 (the rear). Precondition: 0 <= index < size of queue
